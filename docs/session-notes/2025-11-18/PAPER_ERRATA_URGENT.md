## Unified Reinterpretation

### What K-Index Actually Measures

K = 2 √ó |correlation(||obs||, ||act||)|

This measures:
- How proportionally the network responds to input magnitude
- Higher K = more predictable/rigid responses
- Lower K = more context-sensitive/adaptive responses

### Why High K is Harmful

Real tasks require:
- Small perturbations sometimes need large responses (catch falling pole)
- Large perturbations sometimes need small responses (don't overreact)
- Context-appropriate responses, NOT proportional responses

High-K agents respond proportionally:
- Large input ‚Üí Large output (always)
- Small input ‚Üí Small output (always)
- This is MECHANICAL, not INTELLIGENT
